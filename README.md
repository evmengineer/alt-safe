This is a [Vite](https://vitejs.dev) project bootstrapped with [`create-wagmi`](https://github.com/wevm/wagmi/tree/main/packages/create-wagmi).

## Install dependencies

```bash
npm i
```

## Start local server

```bash
npm run dev
```

## Setup

1. Create `.env` file
2. Set property values

## Build

```bash
npm run build
```
